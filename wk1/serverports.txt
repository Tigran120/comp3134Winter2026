1.What is a server port?
A server port is a logical endpoint in an operating system that identifies a specific process or service assigned to handle incoming network traffic.
2.What is the server port range (e.g., 0 to 65535)?
The valid range for server ports is from 0 to 65,535.
3. How many bits does that number represent?
This number represents a 16-bit unsigned integer.
(Calculation: $2^{16} = 65,536$, allowing for values from 0 to 65,535).
